# 常见架构模式的实现

## Pipe-Filter

![image-20201119145756146](image-20201119145756146.png)

+ 适用于数据处理及数据分析系统

+ Filter 封装数据处理的功能

+ 松耦合：Filter 只跟数据（格式）耦合

+ Pipe 用于连接 Filter 传递数据或者在异步处理过程中缓冲数据流

  > 进程内同步调用，Pipe 演变为数据在方法调用间传递
